## Overview

Bluetooth Finder is intended for Windows users who want an easy way to observe nearby Bluetooth and BLE activity. It combines discovery, proximity tracking, and signal-strength analysis so you can see which devices are present, how their status changes, and how near they appear to be.

It fits wireless audits, environment inspections, and ongoing monitoring jobs where clear visibility is important. Thanks to both a desktop interface and a command-line path, it works well for interactive sessions as well as scripted or headless scanning, with OUI lookup, multilingual support, and exportable results for later review.

## Using Bluetooth Finder

How you work with Bluetooth Finder depends on whether you want a visual session or a scripted run:

- Start a scan to discover nearby Bluetooth and BLE devices.
- Review signal strength to understand relative proximity.
- Watch the multi-device dashboard for live changes.
- Apply alert zones when you want to track distance-related thresholds.
- Export results to CSV or JSON for reporting, analysis, or archival use.

Example workflow:

1. Open Bluetooth Finder.
2. Run a discovery pass.
3. Select a device to inspect details and signal history.
4. Save the session output when you need records or comparisons.
5. Use command-line mode when you want repeatable scans or automation.
